# WalletWise - Financial Tracker

A premium, modern financial tracking application to manage your daily income and expenses.

## Features
- **Dashboard**: Real-time view of your total balance, income, and expenses.
- **Visualizations**: Interactive chart showing your income vs. expense ratio.
- **Transaction History**: Keep track of all your spending and earnings.
- **Persistence**: Your data is saved automatically in your browser (LocalStorage), so it's there when you come back.
- **Privacy Focused**: No data is sent to any server. It all stays on your device.

## How to Run
Since this application uses modern web technologies without complex build steps:

1.  Open the folder `financial-tracker`.
2.  Double-click on `index.html`.
3.  The app will open in your default web browser.

## Tech Stack
-   HTML5
-   CSS3 (Glassmorphism, Flexbox, Grid)
-   JavaScript (ES6+)
-   Chart.js (for visualizations)
